# UDS Standard Operation Library

A library for standard operation contracts used in UDS.

## Contracts
- ### Operations
    ```
    ops
    ├ consensus
    │  ├ ProposeOp.sol
    │  └ VoteOp.sol
    └ initialize
        └ InitializeOp.sol
    ```

- ### Predicates
    ```
    predicates
    └ MsgSender.sol
    ```

- ### Utilities
    ```
    StorageLib.sol
    ```
